	#. 반복문
1. While
- 선언부
  while (조건식) {
      내용;
      증감식;
  }


2. For
- for (선언부;  조건식;  증감식) {
      내용
  } 
- 선언부 : 변수를 선언하고 초기화하는 part
- 조건식 : true/false
- 증감식 : eg. i++

- 진행순서
  1. 선언부 -> 조건식 -> 내용 -> 증감식
  2. 증감식 -> 조건식 -> 내용 (반복)
  3. false가 되면 종료
