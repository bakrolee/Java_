	#. 패키지

1. Package
- 의의? : 폴더와 비슷한 개념. 

- 특징
: 소스폴더 안에 만듬.
: 이름은 소문자로 작성.
: 패키지 안에 패키지를 만들 수도 있음. (패키지 안에 있다는 표시로 . 사용) 
  eg. first.second

- 활용
: 클래스를 만들어서 묶어두고 나중에 외부로 보내서 사용가능 (import)


2. 자바에서 클래스를 찾을때
- 같은 패키지에서 찾음.
: 만약 같은 패키지에 없으면 없다고 나타냄.
  -> 이때 쓰는 게 import


3.default or package 접근제한자
- 형태
: 필드 타입 앞에 아무것도 붙이지 않을때
  eg. int number;

- 특징
: 패키지 내에서만 사용 가능 
: 같은 패키지에 있는 다른 클래스들은 접근이 가능하지만, 다른 패키지(그것이 상위 패키지라 할지라도)에서는 접근 불가능.

- 추천
: 가장 엄격한 private을 사용해서 먼저 접근제한을 걸어두고
  이후에 내가 보여주고싶은 범위 내에서 조정하면 됨 (public이든 package 접근제한자든)


	#. 배열
1. 변수 선언
eg.
  int[] aBox; 
  aBox = new int[4];
: 길이가 4인 배열 초기화
: new가 있으니 객체 -> 고로 'aBox = 참조변수'
  (참조 대상 : 4개의 배열을 담을 수 있는 박스)

2. 특징
- 배열은 초기화할 때의 길이값을 변경할 수 없다.(final 성질, 상수)
- 배열의 길이를 초과해서 값을 입력하면 컴파일 에러는 안 생기지만, 실행이 종료됨.


3. 배열과 반복문은 짝이다.
